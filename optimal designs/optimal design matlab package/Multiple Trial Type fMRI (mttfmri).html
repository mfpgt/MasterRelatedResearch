Multiple Trial Type fMRI (mttfmri)
MATLAB Toolbox

This toolbox includes code used in the following papers: Liu and Frank (2004) and Liu (2004). There are also errata available here.

To use the code download the MATLAB *.m files, uncompress, and expand the tar file.  Instructions are in the Contents.m file.  A directory of pre-computed *.mat files is provided here for convenience.  Downloading these is optional. To use them, uncompress and expand the tar file and then move these files into the same directory as the *.m files.
