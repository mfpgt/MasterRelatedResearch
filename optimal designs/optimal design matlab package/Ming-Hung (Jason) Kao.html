Multi-Objective Optimal Experimental Designs for Event-Related fMRI Studies:

We propose an efficient approach to search for optimal designs for event-related functional magnetic resonance imaging (ER-fMRI). Our approach includes well-defined design criteria, an efficient search algorithm that incorporates knowledge about the performance of well-known ER-fMRI designs and a rigorous model formulation. This work will appear in NeuroImage. I have also submitted a paper with a user friendly version of the computer software to the Journal of Statistical Software to provide broad access for practitioners. In addition, we have applied our approach for identifying efficient designs when individual stimulus effects and pairwise contrasts are simultaneously of interest. A paper on this topic has been submitted to the Special Volume of Statistics and Applications in Honour of Professor Aloke Dey. This application again demonstrates the usefulness and flexibility of our approach.

Bayesian Analysis for Multiple Changes of the Long Memory Parameter:

We consider a long memory model, namely ARFIMA-GARCH model, with unknown number of changes in the long memory parameter. We apply a reversible jump Markov Chain Monte Carlo method to estimate model parameters including the number and locations of change points. This work has been presented in 1999 NBER/NSF Time Series Conference, Taipei, Taiwan. An extension of this work using wavelet-based method is proposed by Ko K. and Vannucci M. (2006) IEEE Trans. Sig. Process. 54, 4461-4470.
